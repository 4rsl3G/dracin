# PansaDrama

Minimalist Video Streaming Platform (Node.js + EJS + Tailwind + jQuery SPA).

## Features
- **SPA Experience**: No reload navigation using jQuery & History API.
- **Custom Player**: HTML5 Video, shortcuts, resume playback, auto-next, theater mode.
- **Minimalist Design**: Black & White, Dark Mode default.
- **Proxy API**: Hide upstream API behind Node.js proxy with caching.
